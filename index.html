Tacti'Quatre

Tacti'Quatre est un jeu réalisé dans le cadre d'un projet suivi d'une soutenance. Il s'agit d'un mini jeu de plateau qui mélange les échecs, le puissance quatre et le morpion. En effet, les pions peuvent se déplacer seulement en L comme les cavaliers des échecs. Lors d'un déplacement, le joueur laisse une marque de sa couleur à son emplacement et effectue le déplacement. Fleissner Grid

Fleissner grid est un projet d'arithmétique réalisé par groupe de deux, dans le cadre d'un projet suivi d'une soutenance. Ce projet d'arithmétique a été réalisé en groupe dans le cadre d'un exercice de cryptographie. Il utilise la méthode des grilles tournantes de Fleissner, un outil cryptographique permettant de coder et décoder des messages en positionnant des trous sur une grille puis en la tournant selon un angle prédéfini.
C'est un système complexe, voici donc les fonctionnalités :
Random : Génère une grille de trous aléatoire.
Load/Save : Charge ou sauvegarde la configuration de la grille.
Create : Permet de dessiner les trous manuellement.
Clock : Choisit le sens de rotation (horaire ou antihoraire).
Grid size : Définit la taille de la grille.
Draw : Affiche la grille selon les paramètres.
Clear (zone de texte) : Champ pour saisir le texte à chiffrer.
Cipher : Chiffre le texte saisi et l'affiche.
Decipher : Déchiffre le texte dans le champ "Cipher".
Clear (bouton) : Réinitialise tous les champs.
